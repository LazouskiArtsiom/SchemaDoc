Features

Everything you need to understand your schema

No configuration, no cloud, no data ever leaves your machine. Just run and connect.

⬡

ER Diagrams

Auto-generated entity-relationship diagrams with column types, PK/FK annotations, and relationship arrows. Export as Draw.io for team editing.

⇄

Cross-Database Diff

Compare schemas between Dev, Staging, and Prod in one click. See added tables, removed columns, and type changes at a glance.

↓

Multi-Format Export

Export schema docs as PDF, self-contained HTML, or Draw.io diagrams. Share with your team in the format they prefer.

⌘

Global Search

Press Ctrl+K to instantly search across all tables, columns, views, and procedures. Navigate your schema like code.

🏷

Connection Tags

Label connections as Dev, Staging, Prod, or QA with color-coded tags. Always know which environment you're looking at.

🔒

Private & Secure

Connection strings encrypted with Windows DPAPI, stored locally. Nothing is ever sent to any server. Dark & light mode included.

How it works

Up and running in 60 seconds

No installation wizard, no account, no setup.

1

Download & run

Download the zip, extract it, and double-click SchemaDoc.Web.exe. Your browser opens automatically.

2

Add a connection

Paste your connection string for SQL Server, PostgreSQL, MySQL, or Cosmos DB. It's stored encrypted on your machine.

3

Explore & Compare

Browse tables, generate ER diagrams, export as PDF/HTML/Draw.io, compare schemas across Dev/Staging/Prod, and search everything with Ctrl+K.
